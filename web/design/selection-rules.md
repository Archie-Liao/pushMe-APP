# pushME 设计选型规则表

> 你看 Demo 画廊时填写。不用懂技术，按感觉选即可。  
> **决策** 5 档：KEEP / AVOID / MAYBE / DEFER / CONDITIONAL  
> **强度**：强 / 中 / 弱

## 填表说明

| 列 | 你怎么填 | 我怎么用 |
|----|----------|----------|
| ID | 可省略，我会编号 | 追溯修改 |
| 类别 | 从下方 14 类选 | 知道改哪一层 |
| 决策 | KEEP 等 | 定稿是否包含 |
| 强度 | 强/中/弱 | 冲突时听谁的 |
| 适用范围 | 现代/修仙/全局/聚焦/任务榜/登录 | 别误用到另一主题 |
| 来源 Demo | M2、X1+X3 | 从哪张图来的 |
| 描述 | 你的原话 | 保留意图 |
| 理由 | 为什么 | 比「不要」更有用 |
| 开发提示 | 可选，我来填 | 落成 CSS token |
| 关联 | 如「仅当 R03 时」 | 条件规则 |
