## 跨图共性（现代主题应吸收）

1. **有「产品感」的密度**：不是空灰底 + 两行字，而是侧栏 / 顶栏 / 卡片 / 元数据分层  
2. **颜色有语义**：S/A/B/C 或状态列（紫/黄/蓝/绿），不是全灰  
3. **圆角统一但克制**：12–16px 卡片，按钮 pill 或 8px，**不是** 0 设计  
4. **字阶对比明显**：今日标题大、元信息小、数字/进度可扫读  
5. **任务两种视图**：  
   - **聚焦**：Today / 今日 2–3 件（TickTick、Todoist、KINHIVE Todo）  
   - **榜单/看板**：列表或 Kanban（Todoist 分组、Ramp 列、Notion 看板）  
6. **AI/PM 存在感**：Linear 浮窗、得到大脑右侧对话 —— 符合 pushME「项目经理」  
7. **轻趣味**：Ramp 贴纸、Todoist 小 emoji（克制，非 emoji 堆砌）

## 明确不要（你的反馈 + skill 反 AI 俗套）

- 寡淡灰底、无层次、像「家里死人」的空页面  
- 工程师调试页当设计稿  
- 紫粉渐变英雄区、左边框 accent 卡片、Inter 默认标题  
- 五个方向只换色、不换布局与气质

## pushME 现代定稿方向（待你确认后执行）

**合成策略**（不是抄某一个 App）：

| 模块 | 参考来源 |
|------|----------|
| **暗色方案** | Linear 三栏 + Slack 输入区质感（你的 30-15、30-35） |
| **亮色方案** | KINHIVE 网格 + Todoist 清单结构（47-31、43-53） |
| **聚焦视图** | TickTick Today + KINHIVE Todo 勾选 |
| **任务榜** | Todoist 分组列表 **或** Ramp 看板列（二选一或 Tab 切换） |
| **登录** | Shopify setup 清晰步骤感 / Linear 克制暗色登录 |
| **AI 侧栏** | 得到大脑 / Linear Agent 浮层（阶段 2 可接真对话） |
| **字标** | serif 问候（KINHIVE）+ sans 功能区；`推手` / `pushME` 混排 |

**技能路径**（下次动手必须）：

1. 读 `linear.md` + `notion-pre-ai.md` recipe 数值  
2. `brand-spec.md` 写 token  
3. **先做 1 个高完成度样屏**（登录 + 聚焦 + 任务榜）→ 你验收质感  
4. 通过后再扩修仙 5 或并入 React APP
