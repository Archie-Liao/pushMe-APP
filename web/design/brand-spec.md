# pushME 设计探索 — 品牌与样例数据

> 阶段：Demo 画廊（现代 5 → 修仙 5）

## Logo / 品牌名

- **形式**：文字标（无图形 logo）
- **候选组合**（各 demo 可试不同排版）：
  - `pushME`（英文主标）
  - `推手`（中文主标）
  - `推手 pushME`（中英混排，副标小字）
- **字体**：按方向选用不同 display 字体，**避免 Inter/Roboto 作标题**
- **占位**：无 `[logo]` 方块；直接用文字标

## 反 AI 俗套（来自 web-design-engineer skill）

Demo 与定稿均避免：

- 紫→粉→蓝渐变英雄区
- 彩色左边框 accent 卡片
- Emoji 当图标（修仙休闲文案可用文字语气，不用 emoji 堆砌）
- Inter/Roboto/Arial 作 display 标题
- `#0D1117` 霓虹赛博默认暗色（Linear 暖暗除外）
- 假数据墙、假统计
