# pushME APP — 按计划推进 + 每步在干什么

> 对照 Cursor plan：`pushme_跨端_app_方案_19d0d05a.plan.md`  
> 你是编程小白时，**优先读本文**；技术细节再看 `01-architecture.md`。

## 总目标（一句话）

做一个 **手机 + 电脑都能用的网页 APP**，数据在云上同步，以后能 **AI 催你干活、到点推送**，风格可切 **修仙 / 现代**。

像盖房子：

| 部位 | 技术名 | 大白话 |
|------|--------|--------|
| 门面 + 房间布局 | `web/` 前端 PWA | 你看到的页面、按钮 |
| 仓库 + 保险柜 | Supabase 数据库 | 任务、配置存在云上 |
| 门禁 | Supabase Auth | 只有你能登录自己的数据 |
| 项目经理大脑 | DeepSeek（阶段 2） | 拆解、质疑、复盘 |
| 到点敲门 | Web Push（阶段 3） | 关着浏览器也能弹通知 |

## 计划里的 5 个阶段 — 进度与目的

### 阶段 0：脚手架 ✅ 已完成（你已验收登录成功）

**计划里写什么**：建 Vite+React+PWA；Supabase 项目、表、RLS、登录。

**我们在干什么**：先搭「空房子 + 仓库货架」，还不做完整业务。

| 已做的事 | 文件/位置 | 目的（给你用的） |
|----------|-----------|------------------|
| 建前端工程 | `web/` | 有能打开的 APP 页面，以后功能都加在这里 |
| React + Vite | `web/package.json` | 写界面用的主流工具；`npm run dev` 本地预览 |
| Tailwind | `web/src/index.css` | 快速做好看布局，不用手写大量 CSS |
| PWA 插件 | `web/vite.config.ts` | 以后可加主屏幕、离线缓存、推送基础 |
| 双主题壳 | `web/src/App.tsx` | 现代 / 修仙配色切换（阶段 1 会用在全站） |
| 读密钥 | 根目录 `.env.local` | URL/key 只在你电脑，不进 GitHub |
| 接 Supabase | `web/src/lib/supabase.ts` | 页面能和云数据库说话 |
| Magic Link 登录 | `web/src/App.tsx` | 邮箱收链接登录，不用记密码 |
| 写建表脚本 | `supabase/migrations/*.sql` | 定义 8 张表长什么样 |
| 你在网页执行 SQL | Supabase SQL Editor | **在云上架空柜子**（tasks、profiles…） |
| RLS 权限 | 同上 SQL 文件 | **只你能看自己的任务**，别人登录也看不到你的 |
| Vercel 配置 | `web/vercel.json` | 以后一键部署到公网（尚未连接） |

**你怎么知道成功了**：打开 http://localhost:5173 ，环境已配置 + 邮箱已登录 — **你已经做到了**。

### 阶段 1：核心闭环 ⏳ 下一步

**计划里写什么**：任务/总览/当前焦点 CRUD + 云同步 + 双主题 + 导入任务榜.xlsx。

**我们在干什么**：让你 **真的能管任务**，像 Excel 任务榜搬进 APP，且多设备同步。

| 将要做的事 | 大白话 |
|------------|--------|
| 任务列表页 | 看到 S/A/B/C 任务，像任务榜 |
| 增删改查（CRUD） | 新建、改标题、删、标记完成 |
| 写入 `tasks` 表 | 改完存进云数据库，不丢 |
| 实时同步 | 手机和电脑打开同一账号，看到同一份 |
| 导入 `任务榜.xlsx` | 把你现有 Excel 一次性灌进云 |
| 主题铺满全站 | 修仙/现代不止登录页，任务页也一致 |

**做完你能干什么**：日常用 APP 管任务，不依赖 Excel；换设备登录仍是同一份数据。

---

### 阶段 2：AI 大脑 ⏸️ 未开始

**计划里写什么**：Edge Function 接 DeepSeek；对话收集/拆解/质疑/check-in/评估；对话存档。

**我们在干什么**：APP 里有一个 **会催你、会质疑你的 AI 项目经理**。

| 将要做的事 | 大白话 |
|------------|--------|
| Edge Function | AI 请求走 Supabase 后台，**DeepSeek key 不暴露在浏览器** |
| 对话界面 | 像和 pushME Skill 聊天，但在 APP 里 |
| 写入 `messages` 表 | 你的原文 + AI 精简回复存档（任务榜 A 级要求） |
| check-in / 复盘 | 早晚对齐、做完任务四维评估 |

**做完你能干什么**：在 APP 里被 pushME「推着走」，而不只是在 Cursor 里召唤 Skill。

---

### 阶段 3：主动推送 ⏸️ 未开始

**计划里写什么**：Web Push + Service Worker + pg_cron；到期/停滞任务主动催办。

**我们在干什么**：**关着 APP 也能弹系统通知** 催你（不是微信/钉钉）。

| 将要做的事 | 大白话 |
|------------|--------|
| 订阅通知 | 你点一次「允许通知」 |
| 定时器 pg_cron | 云端每隔一段时间检查「该催谁了」 |
| Web Push | 电脑/手机弹出「pushME：xxx 任务还没动」 |

**限制**：从没打开过 APP、从没允许通知 → 收不到；iPhone 需加主屏幕。

---

### 阶段 4：上架套壳 ⏸️ 以后

**计划里写什么**：Capacitor 套壳 iOS/Android，原生推送，应用商店。

**我们在干什么**：从「网页加主屏」变成 **真正的 App Store / 应用商店安装包**。现在不急。
